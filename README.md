# Shashidhar Reddy Portfolio

This portfolio is a static site built from the resume in `GKSR_RESUME.pdf`.

## Project files

- `index.html` holds the page structure.
- `styles.css` defines the visual system and responsive layout.
- `script.js` contains the portfolio content and rendering logic.
- `assets/GKSR_RESUME.pdf` is the downloadable resume linked in the UI.
